Section Structure

Use the boxes at left to set up the structure for your rubric. It is possible to have up to 5 levels of hierarchy.

The observable parts of your rubric will be used while doing classroom observations. You must select one level in the hierarchy to activate the scoring features. The scoring values defined at left will be used to calculate the scores for calibrations and for organizing the master scores for media.

Rubric Outline

The structure defined in the Master Section above will be used for the rest of the rubric as it is outline to the left. You will be able to enter content into each section while in Edit Mode

You can drag the bars to reorder sections and any content entered during Edit Mode will also be reordered.

If you delete a bar/section then any content entered during Edit Mode will also be deleted. You will see a prompt before anything is removed.
